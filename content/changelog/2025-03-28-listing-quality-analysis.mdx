---
title: Listing Quality Analysis module replaces deprecated tables
description: LQA module replaces product_listing_quality_score tables. Migration guide available.
date: 2025-03-28
tags: [breaking]
---

We launched last year the Listing Quality Analysis module to help you optimize your Amazon product listings with data-driven insights.

It has now officially replaced the following tables, that have been deprecated:
- `product.product_listing_quality_score`
- `product.product_listing_quality_score_breakdown`

To help with this change, check the [migration guide](https://datahawk.notion.site/Listing-Quality-Score-Migration-1ce4aa5f57fa8031bdb9c758e69cc0f2) that outlines how to move to the LQA module and access the latest data ([LQA Module docs](https://datahawk.notion.site/Listing-Quality-Analysis-4b7aff84bea94949bc82f554c6fbfc06)).
