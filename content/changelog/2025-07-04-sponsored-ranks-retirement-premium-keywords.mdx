<Callout type="warn">
**Standard Sponsored Ranks Data is being retired on July 8th** due to quality issues.
</Callout>

- An optional **premium alternative** for Sponsored Keywords is now available as a paid add-on.
- Current subscriptions **continue with all other features** and improvements intact.

**Impacted datasets and dashboards:**

- Search Power BI Dashboard
- Analytics Essentials Looker Studio Dashboard
- `SEO.SEO_PRODUCT_SPONSORED_RANK` table
- Any custom report built on top of this data

**Documentation:**

- [Announcement](https://docs.datahawk.co/sponsored-ranks-tracking-changes)
- [Premium Keywords](https://docs.datahawk.co/premium-keywords)
