title: Connect to Looker Studio
description: Connect DataHawk to Looker Studio via your BigQuery database for live reports and dashboards.
---

Looker Studio is Google's free dashboarding tool. DataHawk connects to Looker Studio via your <Term>BigQuery</Term> database, giving you a live link between your DataHawk data and your reports.

## Prerequisites

Before connecting Looker Studio, make sure you have:

- **A DataHawk <Term>BigQuery</Term> database** — activated in your DataHawk App. If you haven't set one up yet, see [Send data to your tool](/help-center/connect-tools).
- **Data sources connected** — your Amazon and/or Walmart accounts are connected and collecting data. If not yet, see [Set up data](/help-center/data-setup) first.
- **Your Google email registered on the BigQuery access list** — go to **Setup → Destinations → BigQuery → Access** in the DataHawk App and add your email. DataHawk uses this to share access to templates.

## Option 1 — Use a DataHawk template

DataHawk provides a ready-to-use Looker Studio template called **Analytics Essentials**, covering product, keywords, advertising, sales, finance, traffic, and inventory views — all pre-built and connected to your BigQuery data.

To request access:

<Steps>

<Step>
### Open Report Templates

In the DataHawk app, go to [Report Templates](https://analytics.datahawk.co/app/report-templates).
</Step>

<Step>
### Request the template

Find the Looker Studio template and click **Request Access**.
</Step>

<Step>
### Wait for the share

A DataHawk team member will share the template to your Google email.
</Step>

<Step>
### Open the report

You will receive an email invitation — accept it to open the report in Looker Studio.
</Step>

</Steps>

See [Looker Studio Template](/help-center/looker-studio) for the full template guide.

## Option 2 — Connect manually to BigQuery (for custom reports)

If you want to build your own Looker Studio reports from scratch, you can connect directly to your DataHawk BigQuery database using Looker Studio's native BigQuery connector.

<Steps>

<Step>
### Open Looker Studio

Go to [lookerstudio.google.com](https://lookerstudio.google.com/) and create a new report or data source.
</Step>

<Step>
### Add a BigQuery data source

Click **Add data** and select **BigQuery**.
</Step>

<Step>
### Pick your DataHawk project

Select the DataHawk BigQuery project associated with your <Term>workspace</Term>.
</Step>

<Step>
### Choose a dataset or write a query

Choose a dataset (schema) and table, or write a custom SQL query.
</Step>

<Step>
### Connect

Click **Add** to connect.
</Step>

</Steps>

Data in Looker Studio refreshes based on BigQuery's underlying data, which DataHawk updates daily.
