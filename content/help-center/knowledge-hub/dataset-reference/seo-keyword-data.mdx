## Keyword research report tables

All keyword research report tables sit in the `REPORTS` schema.

| Table | Contents | Refresh |
|---|---|---|
| `REPORT_PRODUCT_KEYWORD_SUGGESTIONS` | AI-generated keyword ideas per product | Daily |
| `REPORT_PRODUCT_KEYWORD_LOOKUP` | Organic ranking keywords over the past 7 days | Weekly |
| `REPORT_PRODUCT_KEYWORD_PLACEMENT` | Keyword placement in listings | Paused since Aug 2023 |
| `REPORT_KEYWORD_KEYWORD_SUGGESTIONS` | Keyword-seeded suggestions | On demand |

For full column definitions, use the [Exhaustive Column Referential](https://docs.google.com/spreadsheets/d/1gYGqKQP9Z71nz1lOb4y0wJ15i27NvsUzdliIJeIx0HY/edit#gid=175439716).

## Search volume table

| Schema | Table | Granularity | Refresh |
|---|---|---|---|
| `SEO` | `SEO_KEYWORD_VOLUME_V2` | One data point per keyword per <Term>marketplace</Term> per day (`DATE_DAY`) | Weekly |

<Callout type="info">
  Absence of data for a given date and keyword means that keyword's search volume was below measurable thresholds. It does not indicate a data error.
</Callout>

## Rank tables

| Schema | Table | Contents |
|---|---|---|
| `SEO` | `SEO_PRODUCT_ORGANIC_RANK` | Daily organic rank snapshots for tracked products and keywords |
| `SEO` | `SEO_PRODUCT_SPONSORED_RANK` | Sponsored rank snapshots for Premium Keywords workspaces |

<Callout type="warn">
  Standard sponsored ranks were retired on July 8, 2025. `SEO_PRODUCT_SPONSORED_RANK` is populated only for workspaces using [Premium Keywords](/help-center/data-setup/premium-keywords-tracking).
</Callout>
