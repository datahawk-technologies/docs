DataHawk natively supports Amazon and Walmart data, with new platforms added regularly. If you need a data source we don't yet cover — a vertical-specific marketplace, an internal API, a non-supported retailer, or a custom feed — the **On-Demand Integration Program** lets you request it as a paid add-on. Once built, the new source behaves like any other DataHawk source: warehouse access, dashboards, Alerts, and Insights all included.

## What's already supported

These sources are part of standard DataHawk and don't require an on-demand integration. They split into two groups:

**Account-level connections** — link your existing platform accounts to pull private data (sales, ads, inventory, profit):

- **Amazon** — [Seller](/help-center/data-setup/seller-account), [Vendor](/help-center/data-setup/vendor-account), and [Advertising](/help-center/data-setup/ads-account) accounts
- **Walmart** — [Marketplace](/help-center/data-setup/marketplace-account) and [Walmart Connect](/help-center/data-setup/connect-account) accounts

**Item-level tracking** — add the specific products, keywords, or categories you want a daily snapshot for (works for both Amazon and Walmart, no account connection required):

- [**Products**](/help-center/data-setup/products-tracking) — daily snapshot of attributes, pricing, BSR, ratings, and more
- [**Keywords**](/help-center/data-setup/keywords-tracking) — organic and sponsored rank, plus [Premium Keywords](/help-center/data-setup/premium-keywords-tracking) for richer signals
- [**Categories**](/help-center/data-setup/categories-tracking) — top products and category-level trends

For the full picture of how data flows in and what each source covers, see [How DataHawk Sources Your Data](/help-center/knowledge-hub/how-datahawk-sources-data). New platforms ship regularly — track them in the [Changelog](/changelog).

## When to request a custom source

On-demand integrations cover **additional data that isn't already reachable** through tracked items (Products, Keywords, Categories) or connected accounts (Amazon Seller / Vendor / Advertising, Walmart Marketplace / Walmart Connect). If the data you need can't be pulled by any of those standard paths, this program is how you bring it into DataHawk.
