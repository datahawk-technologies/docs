## What you can monitor

Alerts cover three areas:

- **Listing details** — title, description, bullets, A+ content, images, videos
- **Reviews and ratings** — rating, number of ratings, % critical, % positive
- **<Term>Buy Box</Term> and pricing** — <Term>Buy Box</Term> price, <Term>BSR</Term>, number of sellers, Buy Box owner, listing availability

<Callout type="info">
  Some properties are Amazon-only — see [Triggers & properties](/help-center/modules/alerts/triggers) for the full Amazon vs. Walmart matrix.
</Callout>

## When to use Alerts

Use Alerts when you want to **react quickly to specific changes** on products you care about.

| You want to... | Use Alerts? |
|---|---|
| Be told when a competitor changes price | ✅ Yes |
| Catch a Buy Box loss before it hits sales | ✅ Yes |
| Spot inventory dropping below a threshold | ✅ Yes |
| Discover patterns in your data automatically | Use [Insights](/help-center/modules/insights) instead |
| Understand *why* something changed | Use [<Term>Sherlock</Term>](/help-center/modules/sherlock) instead |

Alerts are reactive: it tells you *that* something changed.

## Key concepts

**Property** — a specific data point you can monitor (e.g. Buy Box price, <Term>BSR</Term>, title).

**Trigger** — a rule that fires when a property changes a certain way. You can use:
- *Preset triggers* — DataHawk's most-used templates (e.g. "losing the Buy Box")
- *Custom triggers* — your own rules built using operators like "decreases by more than 10%"

**Event** — what happens when a trigger fires. Each event becomes part of the daily email.

**Alert** — a saved trigger (or set of triggers) attached to specific products and recipients.

**Recipient group** — a list of email addresses that receive a particular alert. See [Recipient groups & email setup](/help-center/modules/alerts/recipient-groups).

## How alerts get delivered

DataHawk checks your conditions **once a day** and sends **a single daily email per recipient group**, bundling everything that fired in the last 24 hours.

- Delivery is by email
- Alerts data is retained for the **last 3 months**
- Email is sent everyday at 12pm CET

You can also monitor competitors' products — just track them first.
