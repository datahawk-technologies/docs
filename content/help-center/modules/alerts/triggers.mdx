## Preset triggers

Preset triggers are DataHawk's most-used templates. Pick one when you want to get going fast.

| Trigger | What it watches |
|---|---|
| Going out of stock | Stock drops, or status changes from "in stock" to "out of stock" |
| Losing the <Term>Buy Box</Term> | Anyone loses or wins the <Term>Buy Box</Term>, including price |
| New seller on the Buy Box | A new seller starts competing for the Buy Box |
| Entering top 10 <Term>BSR</Term> | Products enter the top 10 in sales rank |
| Dropping out of top 10 <Term>BSR</Term> | Products drop out of the top 10 |
| Becoming a best seller | Product reaches #1 in sales rank |
| Significant price decrease (10%+) | Buy Box price drops by more than 10% |
| Any price change | Any change in Buy Box price (range customisable) |
| Changes in listing details | Any change to the product details page |
| Changes in title | Title text changes |
| Changes in media assets | Number of images or videos changes |
| Changes in star ratings | Increase in % of 1- and 2-star ratings |
| Going offline | Tracked product goes offline |

You can set up as many preset triggers as you like.

## Properties — Amazon vs. Walmart

| Property | Amazon | Walmart |
|---|---|---|
| **Listing details** | | |
| Title | ✅ | ✅ |
| Description | ✅ | ✅ |
| Bullet points | ✅ | ✅ |
| A+ content availability | ✅ | ✅ |
| Number of images | ✅ | ✅ |
| Number of videos | ✅ | ⚠️ No |
| **Reviews and ratings** | | |
| Rating | ✅ | ✅ |
| Number of ratings | ✅ | ✅ |
| % of critical (1–2 star) ratings | ✅ | ✅ |
| % of positive (4–5 star) ratings | ✅ | ✅ |
| **Buy Box and pricing** | | |
| Buy Box price | ✅ | ✅ |
| Sales rank (BSR) | ✅ | ⚠️ No |
| Number of sellers | ✅ | ✅ |
| Buy Box owner | ✅ | ✅ |
| Buy Box availability / status | ✅ | ⚠️ No |
| Listing availability / status | ✅ | ✅ |

⚠️ Walmart parity gaps exist because Walmart's API doesn't expose those data points. If a missing property matters to you, let support know — feature requests help us prioritise.

## Limits and retention

- **Alert history** — retained for the last 3 months
- **Number of alerts per email** — you can see 20 alerts and see more on the DataHawk App
- **Number of recipients per group** — No limit
