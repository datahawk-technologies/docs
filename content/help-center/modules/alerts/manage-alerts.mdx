---
title: Manage alerts
description: Edit, pause, or delete alerts from the Alerts list in DataHawk.
---

Alerts you've already created live in **Apps → Alerts**, where you can review what's firing, tune triggers, or pause noisy rules. This page walks through each option in the order you'd normally use them: find, edit, pause, then delete.

## Find your alerts

Open **Apps → Alerts**. The main list shows every alert in the <Term>workspace</Term>, with:

- Alert name
- Triggers attached
- Products or tags it covers
- Recipient group
- Status (active / paused)
- Last fired

## Edit an alert

1. Click the alert in the list
2. Change any of:
   - Triggers (add, remove, modify)
   - Products or tags it applies to
   - Recipient group
3. Save

Changes take effect on the next daily check.

## Pause an alert

Pausing keeps the alert configured but stops it sending emails. Useful when:

- You're running a planned promotion that would trigger noisy alerts
- You want to investigate a setup without getting flooded
- A product is temporarily out of focus

Click **Pause** on the alert. Click **Resume** when you're ready.

## Delete an alert

If you no longer need an alert, click **Delete**. This removes the alert.

To keep the alert but stop the emails, **pause it** instead.
