AI Copywriter helps you create and optimise Amazon product listings — titles, bullet points, and descriptions — using AI. It generates content that's structured for readability, keyword relevance, and Amazon's listing rules.

<Callout type="warn">
  Available only for Amazon users
</Callout>

## What it does

Pick a product. AI Copywriter looks at:

- Your current listing (title, bullets, description)
- High-potential keywords from DataHawk's Keyword Research
- Amazon's content guidelines and SEO best practices

Then it produces an optimised version side-by-side, with a panel showing which keywords were added, expected traffic gains, and how your version compares.

## When to use AI Copywriter

| You want to... | Use AI Copywriter? |
|---|---|
| Write a listing for a new product | ✅ Yes |
| Improve an existing title or bullets | ✅ Yes |
| Generate variations for A/B testing | ✅ Yes |
| Scale listing creation across a large catalogue | ✅ Yes (with batch limits) |
| Push changes live to Amazon automatically | ❌ No — manual copy/paste into Seller Central needed |

## What you need

- An Amazon **Seller** or **Vendor** account connected to DataHawk
- The product you want to optimise added to your tracked products
