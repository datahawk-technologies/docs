DataHawk <Term>MCP</Term> works with any AI assistant that supports the Model Context Protocol — from chat tools to code editors to automation platforms.

<Callout type="info">
  **Not sure which to pick?** Most users start with **Claude** or **ChatGPT** — they're the fastest to set up (under 2 minutes) and best for ad-hoc questions. Move to a code editor if you're building apps, or n8n / Make if you're scheduling automations.
</Callout>

## Before you start (Prerequisites)

Before connecting any client, set up your DataHawk workspace for MCP:

1. Log in to your DataHawk workspace
2. Navigate to **Settings → Integrations → MCP**
3. Generate your MCP credentials
